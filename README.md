# Node, Express and MongoDB

**UNDER DEVELOPMENT**

## Overview

This is an easy, basic and raw example of **HOW to** setup TypeScript, Eslint, Prettier and Jasmine.

## Requirements

- Node 12+
- NPM

## Install dependencies

```
npm install
```

## Compile TS to JS

```
npm run build
```

### Run the compiled project

```
node build/index
```
